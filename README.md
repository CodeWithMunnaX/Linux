# Linux Commands Ultimate Master Guide 🐧

Welcome to the **Linux Commands Ultimate Master Guide** created by **CodeWithMunnaX**. This repository is designed to help beginners, developers, and system administrators master the Linux command-line interface (CLI) through detailed documentation, real-world examples, and a built-in terminal simulator.

## 🌟 Project Features

- **Interactive Terminal Simulator**: A simulated Bash environment right in your browser where you can execute commands like `ls`, `pwd`, `cd`, `grep`, and get realistic responses.
- **Categorized Directory**: Easily browse commands filtered by categories:
  - 📁 File & Directory Management
  - 📝 Text & Editing
  - ⚙️ System & Resource Monitoring
  - 🌐 Network & Transfer
  - 🛡️ Permissions & Authentication
  - 🛠️ Utilities
- **Detailed Mechanism Explanations**: Deep dives explaining what happens under the hood when a command runs.
- **Real-World Scenarios**: Concrete use cases and shell scripting patterns.
- **Common Pitfalls**: Vital warnings and mistakes to avoid (e.g., recursive deletion traps).
- **Flag Descriptions**: Clean tables showcasing standard flags (`-l`, `-a`, `-rf`, `-p`, etc.).
- **Dual Theme Support**: Beautiful dark mode (default) and premium light mode toggles.
- **Copy to Clipboard**: Quick copy buttons for immediate usage.

## 🚀 How to Run the App

1. Double-click the `Linux_Commands_Master_Guide_Part1_to_Part4.html` file to open it in any modern browser (Chrome, Firefox, Safari, Edge).
2. Alternatively, serve it locally using Python:
   ```bash
   python -m http.server 8000
   ```
   Then navigate to `http://localhost:8000/Linux_Commands_Master_Guide_Part1_to_Part4.html` in your browser.
